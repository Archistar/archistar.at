What is Archistar?

Archistar is a research project funded by the Austrian Federal Ministry for Transport, Innovation and Technology. It is dedicated to the development of a secure distributed storage architecture for trustworthy cloud usage. The Archistar framework combines methods from the domain of Byzantine fault tolerance, cryptographic techniques from secure multi-party computation and additional tools from cloud cryptography into a novel system.

An open-source software prototype is provided. It is capable of connecting multiple cloud resources. This type of configuration is often called multi-cloud paradigm or cloud-of-clouds.

Security Features

Privacy and integrity

Privacy and integrity are guaranteed by the use of robust secret sharing for distributed information. No bounded minority of storage systems have enough information to recover the user's secret. For efficiency Archistar also supports computational secure secret sharing schemes.

Robustness

Byzantine robustness is accomplished by the use of a layer of active cloud components running a modified PBFT protocol. The system can survive arbitrary malicious active attacks from insiders, as long as they compromise less than a third of the system nodes.

Availability

A distributed secure and robust system also increases availability. The system is immune against arbitrary failures in subsets of nodes and therefore can survive outages of single providers.

No vendor lock-in

A vendor and interface agnostic storage layer guarantees high flexibility for remote data storage. Take back your data! Changing providers should be easy and results in monetary savings.
